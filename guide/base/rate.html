# Rate 评分

评分组件

# 基础用法

v-model绑定值，max是评分的数值，同时也控制图标的数量，且必须为整数

<template>
  <mist-rate v-model="rateValue" :max="5"></mist-rate>
</template>
<script>
  export default {
    data() {
      return {
        rateValue: null
      }
    }
  };
</script>

# 禁用

值为：3

disabled组件为只读，show-score开启，在末尾显示评分数值。score-template为显示的模板，会替换{value}。score-color可改变文字颜色。

<template>
  <mist-rate
    v-model="rateValue2"
    disabled
    show-score
    score-template="值为：{value}"
    score-color="#0fb5d0"
  ></mist-rate>
</template>
<script>
  export default {
    data() {
      return {
        rateValue2: 3
      }
    }
  };
</script>
