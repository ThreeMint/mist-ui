# Link 文字链接

文字超链接

# 基础

基础的链接用法

默认链接 主要链接 成功链接 警告链接 危险链接

type控制样式

<template>
  <mist-link href="https://www.bilibili.com" target="_blank">默认链接</mist-link>
  <mist-link type="primary">主要链接</mist-link>
  <mist-link type="success">成功链接</mist-link>
  <mist-link type="warning">警告链接</mist-link>
  <mist-link type="danger">危险链接</mist-link>
</template>

# 禁用

默认链接 主要链接 危险链接

disabled控制文字链接禁用状态。

<template>
  <mist-link disabled>默认链接</mist-link>
  <mist-link type="primary" disabled>主要链接</mist-link>
  <mist-link type="danger" disabled>危险链接</mist-link>
</template>
Expand Copy Copy

# 下划线

无下划线 有下划线

underline控制文字链接下划线，默认为true。

<template>
  <mist-link :underline="false">无下划线</mist-link>
  <mist-link>有下划线</mist-link>
</template>
