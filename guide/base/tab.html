# Tabs 标签页

表示多个相互对立的内容切换。

# 基本用法

选项一
选项二
选项三
选项四
<template>
  <mi-tabs v-model="currentName">
    <mi-tab-pane label="选项一" name="first">选项一</mi-tab-pane>
    <mi-tab-pane label="选项二" name="two">选项二</mi-tab-pane>
    <mi-tab-pane label="选项三" name="three">选项三</mi-tab-pane>
    <mi-tab-pane label="选项四" name="four">选项四</mi-tab-pane>
  </mi-tabs>
</template>
<script>
  export default {
    data() {
      return {
        currentName: 'first'
      }
    }
  };
</script>
