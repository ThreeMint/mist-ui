# 介绍

Mist UI 组件库是一个基于Vue的UI组件库，其中包含了一些场景经常使用的组件，目的是为了减少重复操作，提高开发效率。

# 安装

推荐使用 npm 的方式安装。

npm i mist-ui -S  ||  npm i mist-ui --save
